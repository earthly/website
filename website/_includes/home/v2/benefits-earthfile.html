Earthfiles make defining and managing pipelines easy
We combined some of the best ideas from Dockerfiles and Makefiles into one specification. Most engineers can read an Earthfile instantly, without prior knowledge.
Earthfile
VERSION 0.7
PROJECT my-org/my-project
FROM alpine:3.15
some-pipeline:
PIPELINE
TRIGGER push main
TRIGGER pr main
BUILD +my-build
my-build:
RUN echo Hello world
